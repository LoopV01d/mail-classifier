# Mail Classifier

Проект хакатона по предмету «Технологии программирования».

Программа читает письма из папки `inbox`, классифицирует каждое по ключевым словам и перемещает в соответствующую подпапку внутри `output`.

## Запуск

```bash
bash run.sh
```

Или напрямую:

```bash
python3 src/main.py
```

После запуска папка `output` создаётся автоматически, письма перемещаются из `inbox` по категориям.

## Тесты

```bash
python3 -m pytest
```

## Категории

Программа распределяет письма по восьми категориям: `spam`, `critical`, `access`, `hardware`, `software`, `documents`, `tasks`, `info`. Если письмо не подошло ни под одну — попадает в `unknown`. 
После запуска программы `inbox` остаётся пустым и все письма перемещаются в `output`. Файлы, которые не удалось прочитать (бинарные, пустые, повреждённые), попадают в `output/unprocessed`.
Сначала проверяется тема письма, затем полный текст. Ключевые слова хранятся в `src/keywords.json`.

## Управление категориями(дополнительная функция)

```bash
bash manage.sh
```

Интерактивное меню позволяет добавлять и удалять категории, а также добавлять и удалять ключевые слова внутри них. Все изменения сохраняются в `src/keywords.json` и сразу применяются при следующем запуске `run.sh`.
