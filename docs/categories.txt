Категории:
critical — срочные технические проблемы, из-за которых что-то не работает.
tasks - задачи, которые необходимо выполнить
access — запросы на доступы, права, VPN, корпоративные системы.
hardware — проблемы с техникой: ноутбук, экран, гарнитура.
software — проблемы с программами: Zoom, Excel, Chrome, антивирус.
documents — счета, акты, договоры, оплата.
spam — подозрительные письма, розыгрыши, скидки, фишинг.
info — информационные письма, дайджесты, плановые работы.
unknown — письма, которые не подошли ни к одной категории.
broken — файлы, которые программа не смогла прочитать.

Ключевые слова:
critical:
срочно, urgent, critical, критично, не отвечает, недоступен, ошибка 500, остановлена работа

tasks:
прошу, запрос, просит, подготовить, прислать, уточните, проверить, просим

access:
доступ, права, vpn, active directory, confluence, gitlab, корпоративный портал

hardware:
ноутбук, гарнитура, монитор, экран, клавиатура, мышь, устройство, ремонт, замена

software:
zoom, excel, chrome, adobe reader, антивирус, не запускается, установить

documents:
счёт, счет, акт, договор, оплата, закрывающие документы

spam:
скидка, розыгрыш, выигрыш, банковская карта, перейдите по ссылке, пароль истекает

info:
дайджест, плановые работы, созвон, встреча, статус задач

Порядок проверки:

1. spam
2. critical
3. access
4. hardware
5. software
6. documents
7. tasks
8. info
9. unknown
